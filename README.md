# Student Grade Table
Code for the demo version <a href="https://github.com/ryanthtra/sgt/tree/gh-pages">here.</a>

## Description
This project is a web application that displays a list of students and their grades in certain subjects.  The original MVP required the developer to add and delete entries to a local table.

### Skills learned for the MVP:
>
- Arrays and the manipulation of them.
- Code organizaion and the use of helper functions to assist in the composition of larger ones.
- Using the Twitter Bootstrap library to quickly create and style a responsive page.

### Extra features added post-MVP:
>
- Accessing and manipulating table data on a remote server via AJAX APIs.
- Creating our own databases and APIs with PHP and MySQL.
- Creating gated access to remote server databases with API keys.
